# Syschecker Documentation

This directory contains the documentation for the `syschecker` project, a Go-based system monitoring tool with a Terminal User Interface (TUI).

## Structure

- [Architecture](./architecture/overview.md): High-level system overview.
- [Components](./components/): Detailed documentation for each major component.
    - [UI](./components/ui.md): TUI implementation using `bubbletea` and `lipgloss`.
    - [Collector](./components/collector.md): System metrics collection logic.
    - [Engine](./components/engine.md): Analysis and checking logic.
